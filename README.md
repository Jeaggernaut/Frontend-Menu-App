# Frontend Menu App

Aplicación frontend para gestión de materias hecha con Angular.

## Tecnologías

- **Angular** - Framework 
- **Bootstrap** - Para que se vea bien


## Características

- Autenticación y registro de usuarios
- Gestión de materias
- Sistema de login protegido
- Interfaz responsiva

## Instalación 

```bash
npm install
npm start
```
